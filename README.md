# Telewise Landing Page

Landing page for [Telewise](https://play.google.com/store/apps/details?id=com.telewise), a standalone Telegram client for Wear OS.

**Live:** https://yingwang.github.io/telewise-web/

## Overview

Single-page static site with dark theme. Includes hero section, screenshot gallery with lightbox, features grid, FAQ, and Google Play link.

## Stack

- HTML + CSS (no framework)
- Vanilla JavaScript (lightbox)
- GitHub Pages deployment

## Structure

```
index.html    # Full page
images/       # Screenshots and promo images
```
